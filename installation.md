# Installation

## Requirements

* Yarn

## Installation

Clone the repo:

```text
$ git clone git@github.com:xanyah/xanyah-desktop.git
```

Once you've cloned the repo, enter the directory:

```text
$ cd xanyah-desktop
```

Install dependencies:

```text
$ yarn
```

Start Xanyah in development mode:

```text
$ yarn dev
```
